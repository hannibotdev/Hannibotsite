하니봇 설정 가이드

1. 봇 초대하기

사이트 메인 페이지에서 초대 링크를 통해 서버에 봇을 초대해주세요.

2. 권한 설정하기 (선택사항)

서버 설정에서 봇에게 필요한 권한을 부여해주세요. / 기본적으로 필요한 권한들은 봇을 초대할 때 지급되어 있습니다.

3. 완료!

위에 과정들을 완료하신 후 명령어들을 사용하실 수 있습니다 / 접두사: /

Q: 봇이 응답하지 않아요. 어떻게 해야 하나요?
A: 봇의 상태를 확인하시고, 서버 설정에서 봇의 권한이 올바르게 설정되어 있는지 확인해주세요. / 문제가 지속되면 /버그제보 명령어로 버그를 제보해주세요.
